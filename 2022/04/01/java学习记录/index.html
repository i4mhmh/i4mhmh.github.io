Scanner对象

  • 基本语法:

通过Scanner类的next()与nextLine()方法获取输入的字符串,在读取一串字符串输入之前用hasNext()与hasNextLine()判断是否还有输入的数据.

当输入完毕时,用s.close()关闭

next()与nextLine()的区别在于一个是以空白为停止符,一个是以回车为停止符

方法

System.out.println()

System 是类 out是对象 println是方法

java方法是语句的集合,命名采用首字母小写,后续驼峰法, 方法包含于类或对象中,方法在程序中被创建,在其他地方被调用,一个方法只完成一个功能,保证原子性.

定义方法的格式

  • 修饰符(可选)告诉编译器如何调用该方法
  • 返回值类型: 方法可能会返回值,int char之类的 ,但如果不返回就用void
  • 实参与形参 : 实参 - 实际调用时传递的参数 形参: 定义的时候初始化的参数
  • 若要返回参数 一定要return
  • static证明要直接调用方法
  • Java方法都是值传递

重载

方法名相同,但传递的参数形式不同,这样可以共存在一个类中,Golang中不能重载.

可变参数

方法的参数用…例如 public xxx (int … i) 相当于传进了一个i数组

递归

递归头: 调用的时候需要有一个条件

递归体: 什么时候调用自己

数组

数组是相同类型数据的有序集合, 数组描述的是相同类型的若干个数据,按照一定的先后次序排列组合而成

定义方法

变量类型[] 数组名称

例:

​ 声明数组

​ int[] nums = new int[10]

​ int[] nums

​ 创建数组

​ nums = new int[10]

Array.length 获取数组长度

静态初始化:

​ int[] nums = {1,2,3,4,5}

数组的四个基本特点

  1. 长度是确定的,数组一旦被创建,大小不可以改变
  2. 元素必须是相同类型
  3. 数组中的元素可以是任何蕾数据类型
  4. 数组变量属于引用类型,数组可以看成是对象

数组边界

注意不能越界,便利一般用fori
